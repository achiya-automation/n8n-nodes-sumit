## 🔄 **איך לעדכן (חובה!)**

### שלב 1: עדכן את החבילה

```bash
npm install n8n-nodes-sumit@latest
```

### שלב 2: נקה cache

```bash
rm -rf ~/.n8n/cache
```

### שלב 3: הפעל מחדש

```bash
n8n start
```

### שלב 4: רענן דפדפן

לחץ **Ctrl+Shift+R** (או **Cmd+Shift+R** ב-Mac)
